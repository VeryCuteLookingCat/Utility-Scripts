# Utility Scripts

## Utility.ps1
Windows Utility script that has some QOL features for company infastructure. 
Features:
Feature | Information
--- | --- 
Baseline security policy (auto) | Automatically implements the best Local Security Options policies
Manage user accounts | Manages all users on the system with ease
User rights assignments | Shows all users and groups policy assignments from secpol
